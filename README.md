# Personal Expense Tracker

A Python mini-project that helps users track daily expenses.

## Features

- Add Expense
- View Summary
- View All Expenses
- Update Expense
- Delete Expense
- Save Data Using JSON
- Daily & Monthly Reports
- Expense Trend Chart Using matplotlib

## Technologies Used

- python
- JSON
- Matplotlib

## How to run

```bash
pip install matplotlib
python expense_tracker.py
